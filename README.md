# GiftPro - Premium Gifting & Swag Platform

A modern, responsive landing page inspired by PerkUp for a global gifting and swag solution.

## Features

- **Modern Design**: Clean, professional design with purple and green color scheme
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Interactive elements with smooth scrolling and hover effects
- **All Key Sections**:
  - Hero section with trust badges
  - Premium swag ordering
  - 60,000+ gifts catalog
  - Automation features
  - Client testimonials
  - Blog/resources section
  - Integrations showcase
  - Call-to-action sections

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required - it's pure HTML, CSS, and JavaScript

## File Structure

```
GIFTPRO/
├── index.html      # Main HTML structure
├── styles.css      # All styling and responsive design
├── script.js       # Interactive features and smooth scrolling
└── README.md       # This file
```

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-purple: #6B46C1;
    --primary-green: #10B981;
    /* ... */
}
```

### Content
- Update text content directly in `index.html`
- Replace emoji placeholders with actual images
- Modify company logos and testimonials

### Styling
- All styles are in `styles.css`
- Responsive breakpoints are defined for mobile and tablet views
